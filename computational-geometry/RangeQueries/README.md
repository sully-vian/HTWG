# Practical Assignment 3: Range Queries

- [ ] Implement the 2D-Tree data structure
- [ ] Test the 2D-Tree data structure
- [ ] Implement the interactive construction of the 2D-Tree
- [ ] Implement interactive range search for rectangular query

## Commands

Run these commands, from the current directory.

To generate the `Makefile` run:

```bash
qmake6 All.pro
```

To build the project run:

```bash
make
```

To start the project run:

```bash
./debug/RangeQueries
```

If `compile_commands.json` is needed (clangd LSP support), run:

```bash
bear -- make -j1
```
